Zlzl 正式 MVP 闭环流程图

多模态输入 课堂录音、文档、图片、视频、用户笔记、用户目标
商用处理服务 ASR 转写、文档解析、切片、入库，优先接成熟方案
商用知识库 / RAG 沉淀课程内容、来源、章节、可检索材料，并支撑笔记视图
LLM 学习计划生成 先确定本次学什么、怎么学、完成后验收什么
AI 上下文组装 按学习计划检索知识库，合并 memory 摘要和用户选择
笔记视图 工具性功能：知识图谱视图 + 章节视图
学习 Part 看笔记、章节学习、知识卡片、提问、标记疑问；尽量在产品内发生
验收 Part 词卡、选择题、简答题、口答、语音复述、错因追问
LLM 评估与反馈 评分、点评、错因、薄弱点、下次建议
Memory 写入 保存学习计划、学习行为、作答、反馈、问题和下次建议；MVP 暂不做状态库
用户 选择范围 / 学习 / 验收
课程资料进入产品
工具视图，不参与主循环
按计划调 RAG
按计划组装上下文
完成学习后进入验收
交互证据回写
下一轮继续生成计划
用户参与计划
用户参与学习
用户参与验收
